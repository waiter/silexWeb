接口说明

获取排行榜信息

请求：http://www.waitergame.top/index.php/rank/data/{key}/{phase}?uuid={uuid}
说明：key为排行榜key，必填，phase为第几期，可不填，uuid为用户唯一id，可不填
例子：http://www.waitergame.top/index.php/rank/data/D_8_XX_1000?uuid=1xsda
返回: 
{
  state: 1, // 获取状态，成功为1，msg中是数据，其他情况皆为失败，msg是错误信息
  msg: {
    info: { // 排行榜基本信息
      id: "4",
      gameId: "8",
      name: "dddfgg",
      desc: "dad",
      key: "D_8_XX_1000",   // 排行榜唯一key
      type: "1",
      order: "0",
      length: "100",
      min: "0",
      max: "-1",
      unique: "1",
      pre: "XX",
      num: "1000",
      check: "200",         // 排行榜上传校验值
      updated: "1455717053",
      created: "1455717053"
    },
    phase: 2, // 排行榜第几期
    data: [{  // 排行榜具体内容
      name: "dahu",
      uuid: "1xsda",
      score: "300"
    }, {
      name: "dahu",
      uuid: "2xsda",
      score: "300"
    }],
    user: {  // 如果传了uuid，且找得到该uuid，则有该字段，不然没有
      name: "dahu",
      uuid: "1xsda",
      score: "300",
      rank: "3"
    }
  }
}

上传分数

请求：http://www.waitergame.top/index.php/rank/upload/{key}?uuid={uuid}&name={name}&score={score}&c={check}&force={force}
说明：key为排行榜key，必填；uuid为用户唯一id，必填；name为玩家名，选填；score为分数，必填；check为校验值，必填；force为是否强制更新数据，1为强制，其他或不传都不强制
校验值计算：(int)abs(a*sin(x) + (int)(x/a))，a为排行榜check值，x为分数
例子：http://www.waitergame.top/index.php/rank/data/D_8_XX_1000
返回: 
{
  state: 1, // 获取状态，成功为1，其他情况皆为失败，msg是错误信息
  msg: "上传成功"
}

更改名字

请求：http://www.waitergame.top/index.php/rank/change/{key}?uuid={uuid}&name={name}
说明：key为排行榜key，必填；uuid为用户唯一id，必填；name为玩家名，必填
例子：http://www.waitergame.top/index.php/rank/change/D_8_XX_1000?uuid=xxxhuah&name=waiter
返回: 
{
  state: 1, // 获取状态，成功为1，其他情况皆为失败，msg是错误信息
  msg: "更名成功"
}
